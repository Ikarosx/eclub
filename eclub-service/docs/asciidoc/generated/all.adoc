....

__可选__|string (date-time)
|**Query**|**description** +
__可选__|string
|**Query**|**id** +
__可选__|string
|**Query**|**logoUri** +
__可选__|string
|**Query**|**name** +
__可选__|string
|**Query**|**operator** +
__可选__|string
|**Query**|**regulationUri** +
__可选__|string
|**Query**|**state** +
__可选__|integer (int32)
|**Query**|**updateTime** +
__可选__|string (date-time)
|===

[options="header", cols=".^2,.^14,.^4"]
|===
|HTTP代码|说明|类型
|**200**|OK|<<_responseresult,ResponseResult>>
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
